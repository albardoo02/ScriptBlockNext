# ScriptBlockNext

## 概要
ScriptBlockPlusの主要機能を再現し、軽量化を行ったプラグインです。

また、MythicMobsなど、新たな連携プラグインをサポートしています。

ScriptBlockPlusからの移行機能もあるため、簡単な設定を行うだけでスクリプトを移行できます。

## 備考
- 開発中のため、バグが発生する可能性もあります。
- ScriptBlockPlusにあるすべてのコマンドが実装されているわけではありません。

## 連携プラグイン
- Vault
- PlaceholderAPI
- MythicMobs
- LuckPerms
- DiscordSRV
